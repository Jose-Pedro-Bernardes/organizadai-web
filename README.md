# OrganizaDaí

**OrganizaDaí** é um sistema web de gestão pensado para **prestadores de serviço** que querem organizar seus clientes, serviços, agenda e finanças em um único lugar.  
O objetivo é criar uma plataforma **prática, escalável e fácil de usar**, construída com **Next.js**, **TypeScript** e stacks modernas de front-end.

Este projeto ainda está em desenvolvimento, mas já possui a estrutura base para começar a criar funcionalidades completas de um sistema FullStack.

---

## 🎯 Objetivo do projeto

- Centralizar o gerenciamento de clientes, serviços e atendimentos  
- Controlar agendamentos e compromissos de forma organizada  
- Monitorar finanças e pagamentos  
- Criar uma experiência moderna e responsiva usando **MUI**  
- Permitir integração futura com backend via **Axios** e gerenciamento de estado com **TanStack Query**

---

---

## 📦 Stacks utilizadas

- **Next.js (App Router)**
- **TypeScript**
- **MUI (Material UI)**
- **Axios**
- **TanStack Query**
- **React Hook Form + Zod**
- **CSS Modules**
